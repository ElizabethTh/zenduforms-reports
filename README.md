This project implements the **Cake collections** page based on the provided Figma design as part of a front-end case study.

---

## Features

### UI Implementation

- Header navigation layout
- All Cake collections page based on Figma design
- Search input with filtering
- Add new report button
- Sort dropdown (Newest / Oldest)
- cake list with structured layout
- Pagination controls


## Technologies Used

- Angular (Standalone Components)
- TypeScript
- SCSS
- Angular Material (Icons)
- ESLint (Angular rules)
